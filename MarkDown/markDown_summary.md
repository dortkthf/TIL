## 코드 블록

### fenced code block

- `(backtick) 기호 3개를 활용하여 작성한다 

    ex)  ````python` (띄어쓰기 안해도됨)

- 특정 언어를 명시하면 Syntax highlighting 기능이 적용된다.

```python
print('hello')

if True:
    print('t')
else:
    print('f')
# 주석
```

```html
print('hello')
# 주석?
<h1>
    제목
</h1>
<!-- 주석 -->

```

```javascript
for(i=0;i<10;i++){
}
```
